Setting up the project and get it run!

After you clone or download the source, you can get it run right now with just a little setup work.

Development environment

jdk8 + tomcat8 + eclipse Luna4.4 + chrome.

Create a project for JavaNgArch

  1. In eclipse, create a new Dynamic Web Project, set "Project Location" to the "JavaNgArch" directory.
  2. Set "Source folders on build path" for java to "src".
  3. Set "Content directory" for web module to "web"". Don't use an IDE generated web deployment descriptor.
  4. New a tomcat server to run the project.

Here it is!

Using your own naming

In JavaNgArch， "JavaNgArch" is also used to name a package name and somewhere else. You can easily change it to your own name, like "myapp".

  1. rename the folder /src/javangarch to "myapp".
  2. replace "javangarch" with "myapp" in all .java files under /src/myapp.
  3. In Eclipse,refresh the project.
